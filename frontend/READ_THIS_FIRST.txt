## 🚀 Quick Start (3 Steps)

### Step 1: Install Dependencies
```bash
cd d:\CCS\cca-cw-group-15\frontend
npm install
```

### Step 2: Start Backend (5 services on ports 8001-8005)
```bash
# Each in a separate terminal from project root:
python -m uvicorn backend.identity_service.main:app --reload --port 8001
python -m uvicorn backend.salary_service.main:app --reload --port 8002
python -m uvicorn backend.search_service.main:app --reload --port 8003
python -m uvicorn backend.stats_service.main:app --reload --port 8004
python -m uvicorn backend.vote_service.main:app --reload --port 8005
```

### Step 3: Start Frontend
```bash
npm start
```

**App opens at http://localhost:3000** ✅
